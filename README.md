## Theme

- **Fonts:** Cormorant Garamond (elegant headings) + Quicksand (friendly body)
- **Colors:** Rose, blush, cream, dusty pink, wine accents
- **Extras:** Subtle floating hearts, gradient background, hover states

## Customize

- **Her photo:** Add your image as `public/her-photo.jpg` and in `src/App.jsx` set `HER_PHOTO_SRC` to `'/her-photo.jpg'`.
- **Favorite moments:** Replace the `img` URLs in `FAVORITE_MOMENTS` in `src/App.jsx` with your own photos.

Feel free to change the copy, add a real ‚Äúopen the note‚Äù modal, or turn it into a full Valentine mini-site.
